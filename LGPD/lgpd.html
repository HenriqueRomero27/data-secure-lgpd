Princípios da LGPD

A LGPD é baseada em princípios fundamentais que garantem a proteção de dados pessoais.

  • Finalidade: Os dados devem ser coletados para propósitos legítimos e específicos.
  • Adequação: O tratamento deve ser compatível com a finalidade para a qual foram coletados.
  • Necessidade: Apenas dados necessários para a finalidade pretendida devem ser coletados.
  • Transparência: Os titulares devem ter acesso fácil às informações sobre seus dados.
  • Segurança: Medidas devem ser tomadas para proteger os dados contra acessos não autorizados e vazamentos.
  • Prevenção: Medidas preventivas devem ser adotadas para evitar danos ao titular dos dados.
  • Não discriminação: Dados não devem ser utilizados para discriminação ilegal.
  • Responsabilização: As organizações devem demonstrar conformidade com a LGPD.

Direitos dos Titulares

Os titulares de dados têm os seguintes direitos:

  • Direito de Acesso: Saber quais dados estão sendo coletados e como são usados.
  • Direito de Correção: Solicitar a correção de dados incorretos.
  • Direito de Exclusão: Solicitar a exclusão de seus dados pessoais.
  • Direito de Portabilidade: Solicitar a transferência de dados para outro fornecedor.
  • Direito de Revogação: Revogar o consentimento dado anteriormente.
  • Direito de Informação: Ser informado sobre a lógica do tratamento automatizado de dados.

Obrigações das Organizações

As organizações devem:

  • Implementar Políticas de Privacidade: Ter políticas claras e acessíveis.
  • Nomear Encarregados de Dados: Designar um DPO para supervisionar a conformidade.
  • Registrar Atividades de Tratamento: Manter registros das atividades de tratamentos de dados.
  • Notificar Incidentes de Seguranças: Informar a ANPD e os titulares em caso de vazamento de dados.

Sanções e Penalidades

A LGPD prevê sanções para o descomprimento, incluindo:

  • Advertência: A ANPD pode emitir advertências.
  • Multas: Multas de até R$50 milhões.
  • Suspensão e Proibição: Suspensão do tratamento ou proibição total das atividades de tratamento.
